ng-bind - insert value of specified variable into the tag

<div ng-bind="items.length"></div>

ng-bind-template - insert values into the tag with specified template

<div ng-bind-template="First item: {{items[0].name}}. Second item: {{items[1].name}}."></div>

ng-non-bindable - directive tels Angular not to compile or bind the contents of the current DOM element

<div ng-non-bindable>There are {{items.length}} items</div>
